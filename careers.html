Senior Firmware Engineer

Full-Time · Remote / On-Site

About the Role

We're looking for a Senior Firmware Engineer to architect and implement production firmware for resource-constrained embedded platforms. You'll work across the full firmware stack — from bare-metal peripheral drivers and RTOS task design to high-speed industrial communication protocols — shipping code that runs on hardware in the field.

Requirements

  • Proficiency with modern C++ (C++17/20) on resource-constrained microcontrollers
  • Hands-on experience with TI C2000 and/or STM32 MCU families
  • Experience developing with FreeRTOS (task management, synchronization primitives, memory allocation strategies)
  • Working knowledge of EtherCAT industrial communication protocol (stack integration, slave/master development)
  • Experience offloading work to DMA controllers for high-throughput, low-latency data movement
  • Familiarity with compute hardware accelerators (CLA, CMSIS-DSP, FPU pipelines, hardware math blocks)
  • Comfortable reading schematics and datasheets, and working closely with hardware teams
  • Strong debugging skills — oscilloscopes, logic analyzers, JTAG/SWD

Nice to Have

  • Motor control experience (FOC, six-step commutation, sensorless techniques)
  • PID control system design and tuning for real-time applications
  • Experience with other industrial protocols (CANopen, EtherNet/IP, PROFINET)
  • Contributions to open-source embedded projects

Embedded Controls Engineer

Full-Time · Remote / On-Site

About the Role

We're seeking an Embedded Controls Engineer to design, simulate, and deploy advanced control algorithms on embedded hardware. You'll bridge the gap between control theory and real-time implementation — taking systems from simulation through to production deployment on resource-constrained devices.

Requirements

  • Strong foundation in control theory and practical experience deploying controllers on embedded targets
  • Proficiency with MATLAB/Simulink for system modeling, controller design, and code generation
  • Experience with inverse kinematics for robotic or multi-axis systems
  • Familiarity with modern control techniques including Model Predictive Control (MPC), Reinforcement Learning (RL), LQR/LQG, adaptive control, and state estimation (Kalman filtering)
  • Proficiency in Python for prototyping, data analysis, and control algorithm development
  • Working knowledge of C or C++ for embedded implementation
  • Experience deploying control algorithms on resource-constrained devices with hard real-time requirements
  • Ability to validate controller performance through simulation-to-hardware workflows (SIL/HIL)

Nice to Have

  • Experience with STM32 and/or TI C2000 microcontroller platforms
  • Familiarity with EtherCAT for real-time distributed control
  • Experience with ROS/ROS2 for robotics integration
  • Background in dynamics modeling and system identification
